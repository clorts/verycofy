a categoria

Script Lua — o que é?

Explicação

Lua é uma linguagem de programação leve, rápida e feita para ser encaixada dentro de outros programas — jogos, apps, emuladores e engines. Em vez de rodar sozinha como um app comum, um script Lua é um pedacinho de código que diz a esse programa "faça isso agora", tipo abrir um menu, mudar um valor ou automatizar uma ação.

É por isso que Lua aparece tanto em jogos, mods e ferramentas mobile: ela é pequena, fácil de aprender e não precisa de instalação pesada pra funcionar.

Leve

Roda em qualquer lugar, até no celular, sem travar nada.

Simples

Sintaxe curta. Poucas regras. Fácil de ler mesmo no início.

Encaixável

Vive dentro de outro programa — não é um app sozinho.
